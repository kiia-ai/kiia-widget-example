Client integration example

Embed Kiia explanations in any website.

This static sample shows the smallest production-ready integration: load the Kiia script, provide a container, and call Kiia.explain() with a topic and widget key.

Production script

Replace YOUR_KIIA_WIDGET_KEY in main.js, then generate an explanation.

Minimal integration

Copy this into your site

<script src="https://kiia.ai/embed/kiia.js"></script>
<div id="kiia-container"></div>
<script>
  Kiia.explain("binary search", {
    key: "YOUR_KIIA_WIDGET_KEY",
    container: "#kiia-container",
    type: 1,
    onReady: function () {
      console.log("Kiia explanation ready");
    },
    onError: function (error) {
      console.error("Kiia error", error);
    }
  });
</script>
